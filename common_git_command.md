# Common Git Commands

####  Starting a repo
  - git init


####  Committing Files
  - git add
  - git commit
  - git status
  - git log


####  Managing Committed Files
  - git rm
  - git mv
  - git revert
  - commit SHAs

####  Working with remotes
  - git clone
  - git pull
  - git remote
  - git push
